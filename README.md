# Freelance Trust 

A decentralized milestone escrow platform built on **Stellar Soroban smart contracts**. FreelanceTrust eliminates payment insecurity by securing service agreements between Clients/Employers and Independent Freelance Specialists using transparent on-chain milestone deposits and neutral arbiter dispute resolution.

---

##  Value Proposition

In traditional freelance marketplaces, independent contractors face consistent risks of delayed or withheld payments, while clients risk paying upfront for substandard or incomplete deliverables. 

**FreelanceTrust solves this by:**
* **On-Chain Lockup:** Securing funds transparently in an immutable contract escrow before work begins.
* **Milestone Alignment:** Guaranteeing freelancers immediate programmatic payouts as soon as milestones are verified.
* **Neutral Arbitration:** Protecting both parties via an independent Arbiter address capable of resolving contract deadlocks fairly.
